🌍 TravelQuote PRO – Simulador de Cotización de Viajes

Proyecto final del curso de JavaScript.
Consiste en un simulador interactivo de cotización de viajes que permite calcular el costo estimado de un paquete turístico según diferentes variables ingresadas por el usuario.

✨ Funcionalidades

El simulador permite:

Seleccionar un destino

Elegir categoría de alojamiento

Indicar cantidad de personas

Seleccionar cantidad de noches

Elegir fecha de viaje

Calcular automáticamente el precio total del viaje

Aplicar descuentos según cantidad de noches o personas

Guardar las cotizaciones realizadas en historial

Eliminar historial de cotizaciones

Mostrar los resultados en pantalla de forma dinámica

⚙️ Tecnologías utilizadas

HTML5

CSS3

JavaScript (ES6)

SweetAlert2 para notificaciones

LocalStorage para persistencia de datos

JSON para simulación de datos remotos

🧠 Lógica del simulador

El sistema calcula el precio del viaje considerando:

Precio base del destino

Cantidad de noches

Cantidad de personas

Categoría de alojamiento

Servicios adicionales

Descuentos aplicables

Impuestos

El resultado final muestra un desglose completo de la cotización.

💾 Persistencia de datos

Las cotizaciones realizadas se guardan en LocalStorage, lo que permite:

Mantener el historial de cotizaciones

Visualizar cotizaciones anteriores

Borrar el historial cuando el usuario lo desee
